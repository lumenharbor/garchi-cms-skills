# Agents guide

Garchi CMS is a headless CMS SaaS that facilitates seamless website and app content management.

It is a product by [LumenHarbor Digital Solutions Limited](https://lumenharbor.co.uk)

Before using the skill read the following documents in sequence

1. [Garchi CMS](./garchi-render-content/resources/garchi-cms-doc.md)
2. [Garchi CMS Node SDK](./garchi-render-content/resources/garchi-sdk-node.md) 
3. [Garchi CMS PHP SDK](./garchi-render-content/resources/garchi-sdk-php.md)

**Remember MCP tools are for creating content on Garchi CMS based on user input while this skill is used to render that created content**
